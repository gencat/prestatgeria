
INSTALACIÓN RÁPIDA:

- Descomprime el pack en el directorio web de tu servidor.
- Crea una base de datos MySQL para MyScrapbook. Apunta el nombre de la base y del usuario de esa base.
- Cambia los permisos del directorio "userimages" a 777.
- Cambia los permisos del archivo "config.php" a 777.
- Abre tu navegador web y ve a l directorio donde está tu Scrapbook. Por ejemplo: http://tunombrededominio/myscrapbook
- De ahí en adelante sigue las instrucciones.
- Si quieres información más detallada del proceso, consulta el manual del programa.
